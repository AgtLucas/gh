Node GH GitHub command line tools

Install via NPM

npm install -g gh

Pull Requests

Submit a pull request.

gh pr --submit eduardolundgren

List open pulls requests for the current branch.

gh pr --list

Merge or rebase pull request into a branch.

gh pr --merge [--pull 1] [--branch master]
View more »

Issues

List all issues on the current repository.

gh is --list

List all issues from all repositories.

gh is --list --all
View more »

Notifications

Display the latest repository activities.

gh nt --latest

Watch for any repository activity.

gh nt --watch
